# Pavitra_Webpage

A modern academic portfolio website template for showcasing your CV, publications, patents, and professional links.

## 🚀 Getting started

1. Add your content:
   - Replace placeholder text in `index.html` with your bio, publications, and patent entries.
   - Add your CV as `cv.pdf` (or update the link in `index.html`).
   - Update your LinkedIn / Google Scholar / email links.

2. Preview locally (simple):
   - Open `index.html` in your browser.

3. Publish with GitHub Pages:
   - Push to GitHub.
   - Go to **Settings → Pages** and select the `main` branch and `/` (root).
   - Your site will be available at `https://<your-username>.github.io/<repo-name>`.

## 📄 Files

- `index.html` — your homepage content
- `styles.css` — modern responsive styling
- `cv.pdf` — (optional) place your CV here and link it from the page
